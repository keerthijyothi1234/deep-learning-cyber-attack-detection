# Deep Learning-Based Cyber Attack Detection in IIoT

## 📌 Description
This project focuses on detecting cyber attacks in Industrial IoT environments using machine learning and deep learning techniques. It improves system security by identifying malicious activities in real-time.

## 🛠 Technologies Used
- Python
- TensorFlow
- Scikit-learn
- Pandas
- NumPy

## 🚀 Features
- Achieved 89% accuracy in cyber attack detection  
- Detected 95% of simulated intrusion attempts  
- Built hybrid ML models for classification  
- Data preprocessing, training, and evaluation
